# v0.6.9 — 2026-04-20

## Agent 点对点消息

Agent 之间现在可以用 agent_message 工具互相对话
（发送 / 查询状态 / 等待回复）。发送方 Agent 可自主
选择同步等待反馈或继续执行。

## Uno 统一收件箱

通知、Agent 消息、系统事件，全部汇总到 Uno 的收件箱，
支持按类型筛选和详情展开。该记的都会记下来。

## Agent 自我进化（Beta）

Agent 可以在任务完成后自动优化自己的 prompt 和 skills。
每个 Agent 单独开关；Uno 可代为切换。

## 全局 Token 用量面板

新增 Dashboard 页面，清晰看到每个 Agent 每天的 token
消耗。支持悬停查看明细、按天/总量切换。

## Uno 聊天支持 @ 文件

在 Uno 对话框输入 @ 可以快速引用 workspace 中的文件，
和 Agent 聊天里的体验完全一致。

## ChatGPT 订阅接入优化

全面重做了模型配置测试流程和 ChatGPT 连接面板。修复
了并发请求下偶发的 refresh_token 复用错误。
