# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Clawuno, please **do not** open a public GitHub issue.

Instead, report it privately via one of the following methods:

- **Email:** hello@clawuno.com  
- **GitHub Security Advisories:** [Report a vulnerability](https://github.com/clawuno/clawuno/security/advisories/new)

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Your suggested fix (optional)

We will acknowledge your report within **48 hours** and aim to release a fix within **14 days** for critical issues.

## Supported Versions

We provide security fixes for the latest stable release only. We recommend keeping Clawuno up to date with `clawuno upgrade`.
